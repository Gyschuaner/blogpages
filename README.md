# 个人主页项目

## 项目概述
这是一个使用HTML、CSS和JavaScript构建的个人主页网站，包含个人信息展示、技能介绍、项目经验和联系方式等功能。

## 文件结构
- `index.html` - 主页HTML结构
- `style.css` - 样式表文件
- `script.js` - JavaScript交互逻辑
- `images/` - 图片资源目录
  - `handson.png` - 个人头像
  - `project1.svg` - 项目1图片
  - `project2.svg` - 项目2图片
  - `project3.svg` - 项目3图片

## 如何运行
由于环境限制，您可以通过以下方式打开和测试个人主页：

1. **直接在浏览器中打开**：
   - 找到 `index.html` 文件
   - 右键点击文件，选择「打开方式」
   - 选择您常用的浏览器（Chrome、Firefox、Edge等）

2. **使用浏览器的开发者工具**：
   - 在浏览器中打开页面后，可以按 F12 打开开发者工具
   - 检查控制台是否有错误信息
   - 测试响应式布局（可以使用设备模拟功能）

## 主要功能
- 响应式导航栏，支持移动端菜单
- 个人信息和社交媒体链接展示
- 技能进度条动画
- 项目卡片展示
- 联系表单（带简单验证）
- 返回顶部按钮
- 平滑滚动和页面过渡动画

## 技术栈
- HTML5
- CSS3（使用Flexbox和Grid布局）
- JavaScript（原生JS）
- Font Awesome图标库

## 自定义说明
您可以修改以下内容来个性化您的主页：

1. 个人信息：修改 `index.html` 中的姓名、职业和介绍文字
2. 技能水平：调整 `index.html` 中技能进度条的宽度百分比
3. 项目内容：更新项目卡片的标题、描述和技术标签
4. 联系方式：修改联系信息和表单提交逻辑
5. 样式调整：通过修改 `style.css` 来自定义颜色、字体和布局

## 注意事项
- 所有图片使用SVG格式，保证在任何设备上都清晰显示
- 网站设计遵循响应式原则，适配桌面和移动设备
- 表单提交为前端验证，实际应用中需要配置后端处理逻辑
