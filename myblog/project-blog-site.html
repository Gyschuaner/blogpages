个人博客基地

一个 MC 风格的个人主页与博客站点，用来沉淀 AI、代码、学习笔记和之后真正完成的项目。

项目目标

把个人主页、博客、时间线、技能和项目展示统一成一个有辨识度的小站。视觉上保持方块、箱子、像素边框这些 MC 元素，但阅读内容要比纯装饰更清爽。

已完成

  • 博客页支持按分类箱子筛选文章，并接入打开/关闭箱子图标。

  • 通过 GitHub 仓库连接 Cloudflare Pages 自动部署。
